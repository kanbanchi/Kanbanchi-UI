# Snackbar

Popup with icon, text, button


## variant
> `info` *default*  
> warning  
> success


# Examples
```html
<!-- large buttons must be wrapped with large margins -->

<!-- so as <Button variant="icon">, because of transparent area 48px -->
<ButtonsGroup margin="large">
    <Button variant="icon">
        <Icon />
    </Button>
    <Button variant="icon">
        <Icon />
    </Button>
</ButtonsGroup>

```
